# React.js Typescript application starter

Usage: create fast prototypes, solve test tasks or start a new project with minimal initial setup.

## Tech stack

- [React](https://react.dev/) (v19)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- Tailwind CSS
- **UI Library:** [shadcn/ui](https://ui.shadcn.com/) with Radix UI - accessible UI components built with Radix primitives.
- ESLint
- Prettier

## Quick start

Install dependencies:

```sh
npm install
```

Run dev mode:

```sh
npm run dev
```

Then open http://localhost:5173 page in your browser.

## Create a production build

```sh
npm run build
```

See "dist" folder for results.

To run the production build:

```sh
npm run preview
```

## Type checking

```sh
npm run ts
```

## Linting

```sh
npm run lint
```
